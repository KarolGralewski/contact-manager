# Contact Manager
Contact management tool. Build using Java Spring Boot and React


## Installation

1. To start the backend go to /backend and run
```
./mvnw spring-boot:run    
```
2. To start the frontend go to /frontend and run 
```
npm i 
npm run dev
```
